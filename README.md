# BMI Calculator

## 概要

Pythonで作成したBMI計算アプリです。

ユーザーが身長と体重を入力するとBMIを計算し、その数値に応じた体型判定結果を表示します。

## 作成目的

Pythonの基本的な入力処理、計算処理、条件分岐を学習することを目的として作成しました。

また、健康管理に関する身近なテーマを題材にすることで、プログラミング学習への理解を深めました。

## 使用技術

* Python
* input
* float型
* 計算処理
* if / elif / else
* 比較演算子
* f-string

## 機能

* 身長入力
* 体重入力
* BMI計算
* BMI判定
* 判定結果表示

## 判定区分

* 低体重（痩せ）
* スリムな普通
* 健康的な普通
* がっしり・ぽっちゃり
* 肥満

## 工夫した点

入力された身長と体重からBMIを自動計算できるようにしました。

また、BMIの数値に応じて複数の判定区分を設定し、条件分岐によって適切な結果を表示できるようにしました。

計算結果と判定結果を分かりやすく表示するために、f-stringを使用しています。

## 実行方法

python bmi.py

## 実行例

身長を小数(m)で入力してください: 1.75
体重を小数(kg)で入力してください: 70
あなたのBMIは 22.857142857142858です。
判定結果：健康的な普通
